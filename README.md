# Hosting-website-with-github-pages

This respository contains a simple html and css static websitr deployed using Github pages.

## Steps
1. Created index.html and style.css
2. created a new repo and pushed these codes using Git commands.
3. Turned on GitHub Pages in this repo settings
4. Set the source to main branch and root folder
5. Site is live at : https://soumyalsz.github.io/hosting-website-with-github-pages/
